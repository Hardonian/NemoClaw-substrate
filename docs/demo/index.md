# Demos

The review demos are local and deterministic. They are meant to prove current repository behavior without a GPU, remote worker, or live inference endpoint.

| Demo | Use it for |
|---|---|
| [Local proof](local-proof.md) | Golden path for repository review |
| [Operator walkthrough](operator-walkthrough.md) | Inspecting fixture-backed operator CLI output |
| [Replay walkthrough](replay-walkthrough.md) | Understanding replay rejection and proofpack checks |

Live sandbox operation is covered in the [Quickstart](../get-started/quickstart.md).
