# Operator Truth Model

- Runtime state is authoritative only when backed by receipts/evidence.
- Degraded states must remain explicit, not inferred away.
- Verification claims must map to executable checks.
- Governance claims must map to policy boundaries and deterministic flows.
- No hidden retries, no autonomous daemons, no implicit orchestration.
